# Programa-de-frota-para-taxi-csharp
Um programa escrito em C# cujo função é organizar o lançamento de corridas de uma empresa de taxi ficticia, a NewTaxi.

A NewTaxi é uma empresa cujo objetivo é oferecer um sistema unificado de transporte de pessoas para a cidade de Cubatão-SP, onde a frota é composta por 20 carros e com isso, foi demandado a criação de um programa para controlar o recebimento do dinheiro de cada veículo, e mostrar o carro que mais faturou e o que menos faturou, além de zerar o faturamento de todos os carros no final do dia.
